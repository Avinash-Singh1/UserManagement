# Node.js User Management System - Express, Express-Handlebars, HBS, MySQL



## Create .env file
Create a .env file to store your database credentials

```
DB_HOST = localhost
DB_NAME = yourdb_name
DB_USER = user
DB_PASS = yourpassword
```

## Installation
To run this project, install it locally using npm:

```
$ npm install
$ npm start
```
